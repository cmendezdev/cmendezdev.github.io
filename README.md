# cmendezdev.github.io

Personal portfolio website for Carlos Mendez, hosted via GitHub Pages.

## Stack

- **HTML5** - Semantic, accessible markup
- **CSS3** - Custom properties, responsive layout, no framework
- **Vanilla JS** - IntersectionObserver for scroll-aware nav highlighting

## Local development

```sh
# Serve with any static file server
python3 -m http.server 8000
# or
npx serve .
```

Then open `http://localhost:8000`.

## Design

- Type: DM Sans (Google Fonts) with system monospace for code tags
- Accent: forest green (`#3a7d5a`) on a warm off-white background
- No frameworks, no build step, no dependencies
